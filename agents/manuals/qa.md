# QA 전용 매뉴얼

## 역할

파일이 열리는지, 슬라이드 수·섹션 수가 일치하는지, **Morph 이름 규칙**이 깨지지 않았는지, 대비(다크 배경+밝은 글자)가 유지되는지 검사한다.

## 체크리스트

1. 슬라이드 1 = 갤러리 목차.
2. 슬라이드 2~(N+1) = 섹션별 줌 레이아웃, N = 섹션 수.
3. 선택 창에서 `!!Frame` 접두 객체가 슬라이드마다 동일 개수인지.
4. 전환: Morph가 적용됐는지(첫 장은 이전 없음).
5. 텍스트 오버플로: 썸네일 모드에서 2줄 초과 시 `...` 처리 여부.

## 출력

```json
{ "status": "pass|fail", "issues": [{ "slide": 1, "code": "naming|contrast|overflow", "detail": "..." }] }
```

## 금지

내용의 법적 해석·fact-check는 Research/Editor 범위. QA는 **형식·접근성·템플릿 준수**만 다룬다.
