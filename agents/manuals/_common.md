# 공통 매뉴얼 (짧게 유지)

에이전트는 **이 파일 + 자기 역할 매뉴얼만** 읽는다. 전체 파이프라인 문서를 한 번에 로드하지 않는다.

## 공통 규칙

1. **출력 형식**: 요청된 스키마(JSON / 슬라이드 스펙 / PPTX 빌드 옵션)만 생성한다. 임의 산문은 비본문에 둔다.
2. **근거**: 수치·사실 주장은 출처 또는 “추정” 라벨을 분리한다.
3. **금지**: 개인식별정보, 내부 코드명 사전에 없는 실존 조직/사건을 구체화해 만들어내지 않는다.
4. **브랜드**: 사용자가 정의한 EcoAI(또는 과제) 토큰 색·폰트를 벗어나지 않는다.
5. **도구 경계**: 다른 에이전트 매뉴얼을 대신 해석하거나 덮어쓰지 않는다. 모호하면 오케스트레이터에 질문 스키마로 반환한다.

## 매뉴얼 로딩

- 실행 직전: `_common.md` + `<role>.md` 만 로드.
- 크로스 참조가 필요하면 **요약 한 줄**만 인용하고 원문 전체를 붙여넣지 않는다.
