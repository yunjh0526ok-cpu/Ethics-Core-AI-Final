# Template Design (EcoAI / 다크 갤러리) 전용 매뉴얼

## 역할

**시각 토큰·레이아웃 타입·갤러리 프레임 배치**만 정의한다. 문장 작성은 Report/Editor에 맡긴다.

## 다크 팔레트(기본)

| 토큰 | HEX | 용도 |
|------|-----|------|
| bg.page | `0B0F14` | 슬라이드 배경 |
| bg.elevated | `151B26` | 카드·액자 내부 |
| stroke.subtle | `334155` | 액자 테두리 |
| text.primary | `F8FAFC` | 제목 |
| text.body | `E2E8F0` | 본문 |
| accent | `FDBA74` | 강조·번호 배지 |

## 갤러리 목차(CONTENTS)

- 2행×3열 또는 혼합 가로·세로 **액자** 레이아웃.
- 각 섹션은 **하나의 시각 덩어리**로 보이게: 배경 사각형 + 텍스트 블록(컴포저가 `objectName`으로 묶음).

## Morph 준비 규칙 (필수)

- 액자 배경 도형: `!!Frame{n}_bg` (`n` = 01~06 등 zero-pad 권장)
- 액자 텍스트 상자: `!!Frame{n}_txt`
- 연속 슬라이드에서 **동일 이름·동일 객체 타입**을 유지하고 위치·크기만 변경한다.
- PowerPoint Morph는 `!!` 접두 **선택 규약**; 컴포저가 그대로 반영한다.

## 금지

임의 그라데이션·임의 색 삽입, 라이트 배경으로의 전면 전환(다크 템플릿 요구 시).
